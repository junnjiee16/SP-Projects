My Course and Career Prospect

My Course and Career Prospect

About my Course

I got into Diploma in Applied AI and Analytics (DAAA) through the Early Admissions Exercise (EAE) as I was intrigued about how AI and big data can create huge impact throughout various industries and in the world. This course teaches about data science and how to sieve valuable data to improve products. We also learn how to use big data and Machine Learning techniques to improve existing products or create new and useful applications.

Career Prospect

I aspire to become a software engineer to work in the IT industry as I want to develop and build useful applications that can create a positive impact in the world. There are no limits when working in the field of Inforation Technology and I can get to express my creativity when programming or building applications.
